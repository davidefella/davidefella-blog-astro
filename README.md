# davidefella.dev

Il mio blog personale. Costruito con Astro.

## Setup

```bash
npm install
npm run dev
```

## Scrivere un articolo

Crea un file `.md` in `src/content/blog/`:

```markdown
---
title: "Titolo"
description: "Descrizione"
date: 2026-01-15
tags: ["tag1", "tag2"]
---

Contenuto...
```

## Deploy

Hostato su Vercel, deploy automatico da main.
