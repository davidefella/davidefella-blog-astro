REST (Representational State Transfer) è uno stile architetturale software che definisce un insieme di vincoli da utilizzare per creare servizi web.

I servizi web che si conformano allo stile architetturale REST, chiamati servizi web RESTful, forniscono interoperabilità tra sistemi informatici su Internet.

## I principi fondamentali

I servizi web RESTful permettono ai sistemi richiedenti di accedere e manipolare rappresentazioni testuali di risorse web utilizzando un insieme uniforme e predefinito di operazioni stateless.

Le operazioni più comuni sono:
- **GET** - recupera una risorsa
- **POST** - crea una nuova risorsa
- **PUT** - aggiorna una risorsa esistente
- **DELETE** - elimina una risorsa

## Perché REST è popolare

REST è diventato lo standard de facto per le API web grazie alla sua semplicità e al fatto che utilizza i metodi HTTP standard. Non richiede toolkit costosi o complessi per interagire con i servizi.
