Java è un linguaggio di programmazione ad alto livello, basato su classi e orientato agli oggetti, progettato per avere il minor numero possibile di dipendenze di implementazione.

È un linguaggio di programmazione general-purpose pensato per permettere agli sviluppatori di *write once, run anywhere* (WORA), ovvero il codice Java compilato può essere eseguito su tutte le piattaforme che supportano Java senza necessità di ricompilazione.

## Caratteristiche principali

Le applicazioni Java sono tipicamente compilate in bytecode che può essere eseguito su qualsiasi Java Virtual Machine (JVM) indipendentemente dall'architettura del computer sottostante.

La sintassi di Java è simile a C e C++, ma ha meno funzionalità di basso livello rispetto a entrambi. Il runtime Java fornisce capacità dinamiche come la reflection e la modifica del codice a runtime che tipicamente non sono disponibili nei linguaggi compilati tradizionali.

## Un po' di storia

Java è stato originariamente sviluppato da James Gosling presso Sun Microsystems e rilasciato nel maggio 1995 come componente core della piattaforma Java di Sun Microsystems.
